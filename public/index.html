Commands

  • !motd - display the current DN MOTD
  • !weather <zip,city,location> - tells you the weather in a location.
  • !help - displays this help dialog
  • !btc - returns the current bitcoin price
  • !feature <feature request> - request a feature for the bot
  • !set <setting> <option> - set various settings. <ul>
    • Options: dribbble <username> - set your dribbble username. website <url> set your personal website.
  • !dribbble - return your most recent followed shot (must have !set dribbble <username> before using)
  • !gif <id> - get gif by id from Giphy
  • !gifme <term> - returns a gif related to the term
  • !gifsearch <term> - search for gif id's by term. Returned as PM.
  • !whois <user> - return information set by a user with the !set command
  • !8ball <question> - answers your question with an 8ball.
  • !github - returns the trending github repos for the week.
